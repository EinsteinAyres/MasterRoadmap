# 🧠 PARTE MAIS IMPORTANTE (DIFERENCIAL)

## 🎯 Como responder (framework profissional)

Use sempre:

1. Contexto  
2. Problema  
3. Solução  
4. Trade-offs  
5. Resultado

---

## 🔥 EXEMPLO (RESPOSTA NÍVEL SENIOR)

**Pergunta:** Como você melhoraria performance?

Resposta:

> “Primeiro eu identificaria o gargalo com métricas.  
> Se fosse banco, analisaria queries e índices.  
> Se fosse CPU, revisaria algoritmos.  
> Também consideraria cache e processamento assíncrono.  
> Sempre avaliando trade-offs como consistência vs performance.”

👉 Isso é nível senior.
